# 错误码

| 错误码 | 含义 | 解决方案 |
|--------|------|----------|
| 60020 | IP 不允许 | 检查代理配置是否正确 |
| 48002 | API 被禁止 | 在企业微信后台开启「日程接口权限」|
| 60111 | 用户ID不存在 | 检查 organizer 或 attendees 的用户ID |
| 40058 | 参数错误 | 检查请求参数格式 |
| 41001 | 缺少 access_token | 检查 corpSecret 是否正确 |
| 90482 | op_start_time 不匹配 | 重复日程修改时，op_start_time 必须是某个周期的开始时间 |
| 90484 | 仅修改此日程时不能指定重复 | op_mode=1 时，不能设置 is_repeat=1 |
| 90485 | 非周期性日程不能指定 op_mode | 只有重复日程才能使用 op_mode=1 或 op_mode=2 |
